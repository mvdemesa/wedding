# Images

Put your photos and QR codes in this folder, then reference them in `../config.js`.

Suggested files (names are up to you):

- `hero.jpg` — main invitation background (landscape)
- `story.jpg` — love story background
- `details.jpg` — wedding details background
- `gifts.jpg` — gifts section background
- `sunset.jpg` — footer background (a warm sunset shot looks best)
- `gallery-1.jpg` … `gallery-6.jpg` — the six gallery frames
- `qr-bank1.png`, `qr-bank2.png` — bank QR codes for the gifts section

Tips:
- Keep each image under ~500KB so the page loads quickly. JPG for photos, PNG for QR codes.
- Landscape photos (wider than tall) work best for the full-width backgrounds.
- Any image you don't provide falls back to an elegant gradient placeholder automatically.
